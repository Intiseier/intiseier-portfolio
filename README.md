# inti-portfolio

This repository hosts my personal portfolio website at [intiseier.com](https://intiseier.com).

Built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step. Deployed via GitHub Pages with a custom domain through Cloudflare.

## Tech

- Static site (HTML / CSS / JS)
- [Lucide](https://lucide.dev/) icons
- [PDF.js](https://mozilla.github.io/pdf.js/) for mobile document viewing
- GitHub Pages for hosting
